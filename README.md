# PySnake

A snake game in Python using the Pygame library

How to close: hold 'esc'

## Run

1. Clone the repository using `git clone https://github.com/StefanLenz1/PySnake`
1. cd to the PySnake directory
`cd PySnake`
1. pip install pygame
1. python3 `./snake.py`
